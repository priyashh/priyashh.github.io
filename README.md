# Data Analytics Portfolio – Priyash Vasava


This repository contains my personal data analytics portfolio website. It showcases a few of the projects I’ve worked on and the tools I use as an aspiring data analyst.


## About Me


Hi, I’m **Priyash Vasava**.


I have a strong interest in data analysis and visualization to answer business questions. Most of my work so far uses SQL, Excel, and Power BI to turn raw data into clear insights and visuals.


## Projects on this Site


- **Olist E‑commerce Analysis**
    - Exploratory analysis of Brazilian e‑commerce data.
    - Focus on customer behavior, delivery performance, and revenue trends.
    - Tools: PostgreSQL (SQL), Excel, Power BI.
- **Google Stock Price Analysis**
    - Time‑series analysis of Google’s stock prices and financial metrics.
    - Focus on trends, volatility, and performance over time.
    - Tools: PostgreSQL (SQL), Excel, Power BI.


More projects will be added over time as I continue learning and building new case studies.


## Tech Used for This Website


- HTML, CSS, JavaScript for the portfolio site
- PostgreSQL for data storage and SQL queries
- Excel and Power BI for analysis and dashboards
